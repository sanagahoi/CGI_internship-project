# CGI 2nd Project

This repository contains a front-end admin dashboard web application developed as part of the second CGI project. The project is built using plain HTML, CSS, and JavaScript and is designed to mimic a product administration dashboard with a modern and responsive layout.

## Project Overview

The application provides a simple user interface for managing products and accounts in an admin panel. It includes a login page, a dashboard with charts and summaries, a products page, an add-product form, and an accounts page.

## Key Features

- User login flow with basic validation
- Dashboard with visual statistics and activity sections
- Product listing and management interface
- Add new product form with validation
- Account management page
- Data loading from a sample API response and storage in the browser using localStorage

## Project Structure

- cgi2-templetemo/index.html: Login page
- cgi2-templetemo/dashboard.html: Dashboard view
- cgi2-templetemo/product.html: Product management page
- cgi2-templetemo/account.html: Account management page
- cgi2-templetemo/add-new-product.html: Form for adding a new product
- cgi2-templetemo/*.js: JavaScript files for page logic and interactions
- cgi2-templetemo/style.css and cgi2-templetemo/styles.css: Styling for the application

## How to Run

1. Open the project folder in your browser.
2. Start from cgi2-templetemo/index.html.
3. Navigate through the pages to explore the dashboard and management features.
